\# 🛒 E-Commerce Customer Support Chatbot



AI-powered customer support bot for online stores using Google Gemini.



\## Features

\- 24/7 customer support

\- Product inquiries

\- Order tracking

\- Policy information

\- Custom domain knowledge



\## Tech Stack

\- Python

\- Streamlit

\- Google Gemini AI

\- LangChain



\## Installation

```bash

pip install -r requirements.txt

streamlit run ecommerce\_bot.py

```
